\# AI-Based Employee Performance Prediction System 🚀



This project uses Machine Learning to predict employee performance levels (High, Medium, Low) based on factors like working hours, number of projects, and experience.



\---



\## 📊 Features



\- Data preprocessing using Pandas  

\- Machine Learning model using Random Forest  

\- Model evaluation using:

&#x20; - Precision

&#x20; - Recall

&#x20; - F1-score  

\- Confusion Matrix visualization  

\- User input prediction system  



\---



\## 🛠️ Technologies Used



\- Python  

\- Pandas  

\- Scikit-learn  

\- Matplotlib  

\- Seaborn  



\---



\## 📁 Dataset



The dataset contains employee-related features:



\- Working hours  

\- Number of projects  

\- Experience  

\- Performance (Target variable)
