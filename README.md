# AI-Based Employee Performance Prediction System 🚀

This project uses Machine Learning to predict employee performance levels (High, Medium, Low) based on:
- Working hours
- Number of projects
- Experience

## 🔧 Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib & Seaborn

## 📊 Features
- Data preprocessing
- Random Forest model
- Evaluation (Precision, Recall, F1-score)
- Confusion matrix visualization
- User input prediction

## ▶️ How to Run
```bash
pip install pandas scikit-learn matplotlib seaborn
python main.py
